Auto login account web using Selenium and Python.

In this articale, I will use Python and Selenium library to open a browser to login account.
First, we need something as below:
1. Install Selenium library by using command: pip install selenium.
2. Install Web Driver Server, here I am using Internet Exploer so I install it here: IEDriverServer.exe
3. Install Java Runtime Environment , you should choose the version that match up your computer.

This is the login interface:



SECUREMATRIX is a unique and highly secure. Each time you log in, the number is changed, so the password will change also. Therefore, I make this tool to login automatically regardless of changed number.
For example, my password is below image, it will be 42199803.


We will press F12 to find element of these numbers. The first number is "4" (attribute "alt"="4") so it will be "SMX_BTN_0".

#Initialize password and fill password
password=""
arr = ["SMX_BTN_0", "SMX_BTN_4", "SMX_BTN_8", "SMX_BTN_12", "SMX_BTN_13", "SMX_BTN_14", "SMX_BTN_15", "SMX_BTN_28"]

● Finally, I use try-except to check if there are any errors and inform to user via console output.

finally:
    print("Welcome to the Secure Access SSL VPN, " + driver.find_element_by_class_name("cssSmall").text)
    print("Quit IE.")
    driver.quit()

#Library
from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC
from time import sleep

#User name
userName='a512xxxx'

#Path of IE Driver
driver = webdriver.Ie("D:\Downloads\IEDriverServer.exe")

#Link login
driver.get("https://b2b-gateway.(hidden_name).com/login/ssl")
driver.implicitly_wait(15) 

#Find element and send data
driver.find_element(By.NAME,'LOGIN_ID').send_keys(userName)
driver.find_element(By.CLASS_NAME,'btnCustom').click()

#Initialize password and fill password
password=""
arr = ["SMX_BTN_0", "SMX_BTN_4", "SMX_BTN_8", "SMX_BTN_12", "SMX_BTN_13", "SMX_BTN_14", "SMX_BTN_15", "SMX_BTN_28"]
for i in arr:
    assTemp = driver.find_element(By.ID,i).get_attribute("alt")
    password += passTemp

driver.find_element_by_name("PASSWORD").send_keys(password)
button = driver.find_element_by_class_name("btnCustom").click()
print("Waiting...")

#Check if there are any errors
try:
    element = WebDriverWait(driver, 100).until(EC.presence_of_element_located((By.ID, "table_welcome_2")))

finally:
    print("Welcome to the Secure Access SSL VPN, " + driver.find_element_by_class_name("cssSmall").text)
    print("Quit IE.")
    driver.quit()
