This section is about the projects I have made.

► The door system uses fingerprints

Used language: C/C++ using compiler Arduino (Jul 2019 - Sep 2019)

- Using Arduino, fingerprints module and electric lock.
- The system will read the valid fingerprint or not to decide to close/ open the door via an electric lock.
- Besides that, the user can add or delete fingerprints if necessary.

► Power management

Used language: C/C++, PHP (Dec 2018 - Aug 2019)

- Using PZEM004T module (power measurement module), Arduino and LCD (version 1), ESP32, and web (version 2).
- The system will measure the parameters such as voltage, current, power and energy of electrical devices.
- All information will be displayed on LCD or web. So, the user can track and optimize the usage to avoid waste.

► Control devices via Radio Frequency (RF) waves

C/C++ using compiler Arduino (Jul 2019 - Sep 2019)

- Using Arduino, 433MHz RF Transmitter and Receiver Module.
- The signal will be processed (read, analysis Arduino, then turn on/off devices (such as the lamps, fan) via Optotriac.
- It can control the devices at a maximum distance of 20 meters.
